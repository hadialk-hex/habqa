## ⚠️ تنبيهات مهمة

1. **الإصدار الحالي في المشروع `v21.0` منتهي الصلاحية** — يجب الترقية فوراً إلى `v25.0`
2. **شهادة mTLS جديدة** — اعتباراً من مارس 2026، يجب قبول شهادة Meta CA الجديدة
3. **مقاييس محذوفة** — تم حذف Post/Page Reach و Video Impressions في يونيو 2026
4. **سياسة الروبوتات** — اعتباراً من يناير 2026، الروبوتات "ذات الأغراض العامة" ممنوعة على WhatsApp
